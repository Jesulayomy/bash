### [Bestday](bestday.sh)
Usage
```commandline
./bestday.sh
```
Or add to .bashrc on a newline
```commandline
bash /path/to/dir/bestday.sh
```
Tells the machine to run with bash. This file can be configured at startup to give you a good day start.
